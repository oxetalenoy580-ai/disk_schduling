# Disk Scheduling

A disk scheduling algorithm visualization tool built with TypeScript and HTML Canvas.

## Algorithms

- [ ] FCFS (First-Come, First-Served)
- [ ] SSTF (Shortest Seek Time First)
- [ ] SCAN
## Usage

Open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```

Click the algorithm buttons to visualize seek patterns on the canvas.

## Build

```bash
npx tsc
```
